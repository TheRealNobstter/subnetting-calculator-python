# IPv4 Subnetting Calculator

A Python-based IPv4 subnetting calculator with a graphical user interface (Tkinter).

---

## 🇬🇧 Description

This application allows users to calculate subnet information based on a given IPv4 address and subnet mask.

It supports:
- RFC 950 (classic subnetting: 2ⁿ − 2)
- RFC 1878 (modern subnetting: 2ⁿ)

### Features
- Input of IP address and subnet mask (CIDR or dotted decimal)
- Automatic class detection (A/B/C)
- Calculates:
  - Network address
  - Broadcast address
  - Host range
  - Number of subnets
  - Number of hosts
- Displays all subnets (with limit for large outputs)
- Highlights the subnet containing the given IP
- GUI built with Tkinter
- Animated output (line-by-line display)
